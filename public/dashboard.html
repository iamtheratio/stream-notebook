Step 1 · Connect Twitch

Links your Twitch account so the notebook can read !note commands from your chat, post confirmations back, and follow whatever game your channel is set to. You approve it on Twitch — nothing to copy or paste here.

Not connected
Click connect and approve the code on twitch.tv.

Step 2 · Add the overlay to OBS

In OBS: Sources → + → Browser. Paste this URL, set the size to 1920 × 1080, and tick Shutdown source when not visible. The notebook stays hidden until a note is added or you run !note show.

Step 3 · Try it

Type !note hello in your own Twitch chat. It should pop up on the overlay within a second. That’s it — you’re set up. Everything below is optional and already set to sensible defaults.

Options

All optional. Saved automatically, and they apply straight away — no restart.

Reply in chat
Posts short confirmations like “📓 Note #3 added.” Turn off for a quieter chat — the overlay still updates.
Organise notes by game
Auto follows your Twitch category, so each game keeps its own notebook. Manual uses one fixed name for everything.
Notebook name
All notes are filed under this name.
Moderators can manage notes
Full control — add, edit, mark done, delete and clear. Turn this off if you’d rather be the only one who can change the notebook.
VIPs can add notes
Adding only — they can’t edit, delete or clear.
Subscribers can add notes
Adding only. On a big channel this is a lot of people — start it off and see how it goes.
You always have full control, whatever these are set to. Everyone else can still use !note show, hide and list.
Viewer cooldown
Seconds a regular viewer must wait between !note show / hide / list. Stops people spamming the overlay on and off. 0 turns the wait off.

Overlay Controls

Same as the chat commands, for when you’d rather click than type.

Starting & stopping

Start automatically when I turn on my PC
The notebook starts by itself and waits, minimised, in your taskbar. Recommended — otherwise you have to remember to start it before every stream.
Stop the notebook
Turns it off completely. Your notes are saved and will still be there next time. To start it again, double-click start.bat.
